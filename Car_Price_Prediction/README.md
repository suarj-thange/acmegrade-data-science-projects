# Car Price Prediction

This project predicts car prices based on features such as model, year, mileage, transmission, fuel type, and engine size.

## Objective
To build a regression model to accurately predict car prices.

## Models Used
- Linear Regression
- Random Forest Regressor
- Extra Trees Regressor
- CatBoost Regressor

## Outcome
CatBoost Regressor provided the best performance with high accuracy and low error.
